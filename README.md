# Frameworks

A combined branch that collects reference implementations and study notes for different Pydantic- and LLM-focused exercises i implemented through their documentation and reference articles

## Contents

- `Pydantic` – quick experiments with classic Pydantic models and Python typing patterns.
- `Pydantic-AI` – exploratory notebooks and transcripts using the upcoming Pydantic AI agent framework.
- `GPT-2+Pydantic` – full-stack sample combining a FastAPI backend with a React frontend around a fine-tuned GPT-2 storyteller.
